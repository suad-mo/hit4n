Help Description Example Hint

The hit4n game is a game of guessing an unknown sequence of four different digits.

At the beginning of the game, a random string of four different digits is generated.

Numbers from 0 to 9.

Enter 4 digits after which we get the answer in the form X : Y =>

X - total number of hits

Y - how many of them are in place.

The game continues until we hit all four numbers and their positions

Hints

In each row we can check one number by clicking on it

Clicking on a number in a series of unknown numbers will change the background
